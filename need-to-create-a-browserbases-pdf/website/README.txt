Mahesh PDF Editor Website

For local computer use, double-click:

start-local-website.bat

Then open:

http://localhost:4185/index.html

Opening index.html directly may not load the visual PDF preview in some browsers.

Upload both items together:
- index.html
- vendor folder
- start-local-website.bat

Features:
- Merge PDFs
- Split selected pages
- Split every page into separate PDF downloads
- Add new text to a PDF page and download the edited copy
- Preview the PDF page while dragging the new text into position
- Delete existing visible text by covering it with a white box
- Change existing visible text by covering it and placing replacement text over it

Text editing note:
The Edit Text tool works visually. It covers existing text with a white box and can place replacement text on top. It does not rewrite the PDF's internal text layer like a Word document.

Google Drive note:
Google Drive can store this file, but it is not a normal public website host. If you upload index.html to Drive, visitors may see a preview or download page instead of a live website.

Best free hosting options:
- GitHub Pages
- Netlify
- Cloudflare Pages

If you specifically want to keep it in Google, use Google Cloud Storage static website hosting instead of Google Drive.
